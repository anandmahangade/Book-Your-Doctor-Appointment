# 🩺 Book Your Doctor Appointment

A simple web-based **Doctor Appointment Booking System** that allows patients to book appointments with available doctors easily. The application provides a clean user interface for selecting doctors, choosing time slots, and submitting appointment details.

## 🚀 Features
- Book appointments with specialist doctors
- Select preferred date and time slots
- View doctor profiles and specialties
- Patient testimonials section
- Like/favorite doctors
- Light/Dark theme toggle
- Responsive UI using Bootstrap

## 🛠️ Tech Stack
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
